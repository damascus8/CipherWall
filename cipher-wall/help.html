📘 How to Use Cipher Wall

👋 Welcome to Cipher Wall — Your Personal Message Shield

Cipher Wall is a secure, offline tool for encrypting, decrypting, storing, and sharing your sensitive text and image data.

──────────────────────────────────────────────
🧩 Features Covered

🔐 TEXT ENCRYPTION/DECRYPTION
- Supports: AES, Base64, ROT13, Caesar Cipher, Morse Code
- Input your message and (optionally) a key
- Works instantly in the browser

🖼️ IMAGE ENCRYPTION/DECRYPTION
- Select any image (PNG, JPG, etc.)
- Set a strong password
- Image is encrypted and uploaded securely
- Generates a downloadable encrypted file + QR link for retrieval

📤 QR CODE GENERATION
- Generate QR codes for encrypted or plain messages
- Shareable and scannable via mobile
- QR links redirect to a safe `message-view.html` page with decryption options

📥 QR SCANNING & VIEWING
- Scan generated QR using your device
- Auto-loads the encrypted content into a secure reader
- Prompted to enter the password/key (if required) to decrypt

📄 DOWNLOAD FUNCTIONALITY
- Download input/output messages as `.txt` files
- Encrypted images can also be downloaded locally
- Secure file naming based on timestamp

🧾 ACTION LOGGING
- Auto-generates and appends actions in an on-screen log
- Useful for tracing your encryption/decryption activity

🌐 URL-BASED MESSAGE SHARING
- Encrypted messages are hosted with unique IDs
- URLs redirect to viewer with a pre-filled data field
- Only those with the URL + key can decrypt

🧠 TIPS FOR BEST USE
- AES & Caesar require a key — choose wisely
- ROT13, Base64, and Morse don’t need a key
- Use numeric keys for Caesar (e.g., 3)
- Morse uses `/` for spaces
- Never reuse sensitive passwords across services

🔒 PRIVACY PROMISE
- 100% client-side operations for text
- No text data is sent to any server unless using hosted QR link
- Image encryption sends data securely to backend for temporary storage
- All encrypted files are safe and non-readable without your password
